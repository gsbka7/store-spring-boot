## 2. 功能

1. 註冊 (驗證是否已被註冊、內容是否填寫完整)

2. 登入 (驗證帳號是否存在、密碼是否正確)

3. 購物車相關功能 (登入後才可加入商品、加入商品、購物車頁面商品展示、金額數量計算)

4. 建立訂單

5. 查詢訂單紀錄與詳情

## 3. 部署

#### 網址：[https://store-spring-boot.onrender.com/](https://store-spring-boot.onrender.com/)
